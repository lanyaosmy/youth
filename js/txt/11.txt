			<h1>关于组织开展四川大学大学生法治知识竞赛的通知</h1>
			<div class="attr">
				发布时间：2014-05-15 18:38:00  浏览次数：<script src="//www.google-analytics.com/analytics.js" async=""></script><script src="/index.php/system/web/view/viewnumber/id/5165" type="text/javascript"></script>172
			</div>
			<div class="content-text">
				<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;">全校同学、各学院团委：</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">为深入贯彻《中共中央关于全面深化改革若干重大问题的决定》和《四川省依法治省纲要》，弘扬“法治中国”理念，执行依法治校方针，积极培养大学生“自由、平等、公正、法治”的社会主义核心价值观，引导大学生知法、懂法、守法、用法，在大学生中普及、增强法治意识，结合学校党风廉政建设宣传月工作要求，校团委联合法学院于本学期开展了学生系列法治主题教育活动，四川大学大学生法治知识竞赛是该系列活动中的重要环节，也是“挑战杯”四川大学2014年学生科技节的重要赛事，该项竞赛由商学院团委、法学院团委、校学生学术科技协会具体承办，现将有关事项通知如下。</span></span></div>
<div>
	&nbsp;</div>
<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>一、参赛对象</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>本次竞赛分为个人赛和团体赛，个人赛分为专业组和非专业组，分别进行设奖。</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>1</strong><strong>、个人赛参赛对象：</strong>全日制在校学生（包括本科、研究生），专业组（法学院全日制在校学生），非专业组（除法学院以外的，其他学院全日制在校学生）</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>2</strong><strong>、团体赛参赛对象：</strong>除法学院以外的，其他学院全日制在校本科学生，参赛团队包括：学院代表队和跨学院团队，每支团队人数为6人。</span></span></div>
<div>
	&nbsp;</div>
<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>二、比赛时间进度：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>1</strong><strong>、报名时间：</strong>自主报名仅限于个人参赛和跨学院组队团体，参赛选手可兼报个人赛和跨学院团队赛，即日起至5月21日20：00。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>2</strong><strong>、青春广场现场报名和互动环节：</strong>5月17日、18日</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>3</strong><strong>、学院代表队选拔时间：</strong>各学院在5月21日之前完成选拔工作，每个学院仅限一支代表队参赛。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>4</strong><strong>、个人赛</strong><strong>开展时间：</strong>5月24日</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>5</strong><strong>、团体赛初赛开展时间：</strong>5月25日</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>6</strong><strong>、团体赛半决赛开展时间：</strong>5月31日</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>7</strong><strong>、决赛开展时间：</strong>6月6日</span></span></div>
<div>
	&nbsp;</div>
<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>三、具体赛程安排和内容简介</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>1</strong><strong>、青春广场现场报名和互动环节：</strong>5月17、18日9：00—18：00，工作人员将在青春广场设置宣传点，进行法治知识现场问答互动，并接受个人参赛或跨学院组队参赛的现场报名。活动现场设有精美的小礼品。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>2</strong><strong>、个人赛</strong><strong>：</strong>分为专业组和非专业组，于5月24日以笔试形式开展。工作人员将在5月21日报名结束后，向报名参加个人赛的选手统一发布参赛题库，个人赛笔试范围不仅限于题库内容。比赛具体时间、地点另行通知。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>3</strong><strong>、团体赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>初赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">工作人员将在5月21日报名结束后，向报名参加团体赛的团队统一发布参赛题库，团体赛笔试范围不仅限于题库内容。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">5月25日，由各学院代表队和跨学院团队进行团体赛初赛，初赛以笔试形式开展。赛事组织方将根据初赛成绩选拔12支成绩最优的团队晋级团体赛决赛。比赛具体时间、地点另行通知。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>半决赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">5月31日，晋级半决赛的12支团队进行现场角逐，比赛形式包括个人必答题环节、团队必答题环节、抢答环节、风险题环节、案例分析题等竞赛环节。赛事组织方将根据半决赛成绩选拔6支成绩最优的团队晋级团体赛决赛。比赛具体时间、地点另行通知。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>决赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>在团体决赛开展前，赛事组织方将安排参赛团队集中培训，培训时宣布决赛环节具体要求。</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">6月6日，晋级决赛的6支参赛团队进行现场角逐，每支参赛团队3名同学上场参赛（各团队需在6月4日前确定上场参赛人员，并报工作人员备案）。比赛形式包括个人必答题环节、团队必答题环节、抢答环节、风险题环节、案例分析题等竞赛环节。比赛具体时间、地点另行通知。</span></span></div>
<div>
	&nbsp;</div>
<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>四、</strong><strong>报名方式</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>1</strong><strong>、线上报名</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>线上报名仅限于跨学院组队和个人参赛两种形式，</strong>请有意参加的跨学院团队和个人于5月21日20：00之前填写报名表（见附件），并发送至指定邮箱： &nbsp;<a href="mailto:scu2013xt@163.com%EF%BC%8C%E9%82%AE%E4%BB%B6%E5%91%BD%E5%90%8D%E4%B8%BA"><span style="color:windowtext;">scu2013xt@163.com，邮件命名为“法治知识竞赛</span><span style="color:windowtext;">+</span><span style="color:windowtext;">个人</span><span style="color:windowtext;">/</span><span style="color:windowtext;">跨院</span><span style="color:windowtext;">+</span><span style="color:windowtext;">姓名</span><span style="color:windowtext;">/</span><span style="color:windowtext;">队长姓名”。</span></a></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">2、<strong>线下报名：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>个人赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">5月17、18日9:00—18：00青春广场接受现场报名</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>团体赛：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">学院代表队：于5月21日前由学院选拔产生，每个学院一支代表队。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">跨学院组队：5月17、18日9:00—18：00青春广场接受现场报名。</span></span></div>
<div>
	&nbsp;</div>
<div>
	<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>五、</strong>&nbsp; <strong>赛事奖励：</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>获奖个人和团队将获得奖状。</strong></span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>个人赛：</strong>专业组和非专业组分别设奖，原则上各组参赛人数在500人以下，获奖比例约为30%；各组参赛人数在500人—1000人，获奖比例约为25%；各组参赛人数超过1000人，获奖比例约为20%。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>优秀组织奖：</strong>个人赛参赛人数最多的10个学院将获得优秀组织奖。</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>团体赛：</strong>团体一等奖2名</span></span></div>
<div>
	　　　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 团体二等奖4名</span></span></div>
<div>
	　　　　<span style="font-size:16px;"><span style="font-family:微软雅黑;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 团体三等奖6名</span></span></div>
<div>
	　　<span style="font-size:16px;"><span style="font-family:微软雅黑;"><strong>竞赛最终解释权归组织方所有。</strong></span></span></div>
<div>
