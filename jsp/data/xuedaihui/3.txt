<div class="content-art">  			<h1>关于成立四川大学第三十次学生代表大会筹备工作委员会的通知</h1>  			<div class="attr">  				发布时间：2017-04-25 09:17:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/13833"></script>  			</div>  			<div class="content-text">  				<p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">各学院学生会：</span></span></p> <p align="left" style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">距四川大学第二十九次学生代表大会召开已近两年。两年来，在校党委的正确领导、校团委的悉心指导和上级学联的带领下，四川大学学生会秉承&ldquo;全心全意为同学服务&rdquo;的宗旨，紧密联系同学，在服务同学精神成长、学习生活、成才发展、权益维护等方面取得了突出成绩。为深入贯彻落实习近平总书记系列重要讲话特别是关于青年工作、群团工作和共青团工作的重要指示精神，贯彻落实《学联学生会组织改革方案》《共青团中央改革方案》《高校共青团改革实施方案》及有关文件要求，拟于2017年6月召开四川大学第三十次学生代表大会，以强化学生会组织&ldquo;自我教育、自我管理、自我服务、自我监督&rdquo;的职能，增强学生会组织的政治性、先进性和群众性。</span></span></p> <p align="left" style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">经研究，决定专门成立&ldquo;四川大学第三十次学生代表大会筹备工作委员会&rdquo;（以下简称&ldquo;筹委会&rdquo;），欢迎全校同学报名参加。现将有关事项通知如下：</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;"><strong>一、机构设置</strong></span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">&ldquo;筹委会&rdquo;日常工作由筹委会领导小组主持，领导小组下设秘书工作组、代表工作组、提案工作组、会务工作组、宣传工作组、选举工作组六个职能工作小组。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">1.秘书工作组：负责协调各组运作。负责筹委会内部的通知联络；负责大会主要文案的起草、印制；负责各次会议主持词、文案材料的校订；负责临时会议材料的起草、印制；负责各次会议的内容录音、记录工作；负责大会物品的采购及管理；</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">2.代表工作组：负责服务学代表履职。确定学代表的名额划分；负责监督各选举单位选举代表；负责对代表进行资格审查、名单公示；负责编制代表名册，划分代表团；负责学代表的活动及参会通讯联络；负责学代表证件的印制以及其他物资的分发。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">3.提案工作组：负责学代表提案的收集。学代表提案的整理及统计；优秀提案的评选及推广；负责学代表提案的答复。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">4.会务工作组：负责大会会务设计及规范。负责主持稿及附件材料的起草，经秘书工作组校订后印制；负责大会会场及分团讨论地点的联系、布置、协调；负责校内单位、兄弟高校的邀请及接待；负责车辆协调、礼仪组织等服务工作；负责大会召开期间签到、人数清点及报告；负责大会召开期间的安保工作。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">5.宣传工作组：负责大会相关宣传。负责大会背景、会务手册及其他相关设计；负责大会合照的联络、组织；负责大会召开期间的摄影、摄像；负责大会召开期间的简报；负责大会宣传画、宣传标语、宣传片的制作和推广；</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">6.选举工作组：负责第三十届第一任学生会主席团的选举工作。负责换届通知的起草；负责笔试、面试考评、主席论坛、委员会选举、主席团选举等环节的组织和记录；负责选票及其他附件材料的印制。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;"><strong>二、报名条件</strong></span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">1.具有坚定的政治立场和政治方向，具备良好的理论素养和政策水平。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">2.在校期间无违反校纪校规行为，专业成绩良好，综合表现优异。</span></span></p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-size:18px;"><span style="font-family:仿宋_gb2312;">4.熟悉学联学生会工作理论及实践的学生会骨干优先。</span></span></p> <p style="margin: 25px 0px;
