<div class="content-art">  			<h1>关于推选四川大学第三十次学生代表大会正式代表的通知</h1>  			<div class="attr">  				发布时间：2017-04-25 09:20:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/13835"></script>  			</div>  			<div class="content-text">  				<div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">各学院：</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">为加强四川大学学生代表大会（以下简称&ldquo;学代会&rdquo;）制度建设，充分发挥学生代表大会在团结广大同学、促进成才发展、维护同学权益、参与校园治理等方面的重要作用，充分保证学代会的政治性、先进性和群众性，保证学代会代表（以下简称&ldquo;学代表&rdquo;）的广泛性和代表性，根据《学联学生会组织改革方案》的相关要求，按照《四川大学学生会章程》的相关规定，现就本次学代表选举的相关工作通知如下：</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>一、学代表的基本条件</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">1.正式注册的四川大学全日制本科在校生；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">2.具有坚定的政治立场和政治方向，具备良好的理论素养和政策水平，作风正派，诚信守法；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">3.在校期间无违纪违规行为，专业成绩良好，综合表现优秀；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">4.具有责任心和奉献精神，关注学校发展和同学权益，有一定的参政议政能力，拥有良好的群众基础和影响力。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>二、学代表的权利和义务</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">学代表享有下列权利：参加学生代表大会，参与审议各项提案、报告和其他议题；联名提出提案；参加学代会的各项选举、参加学代会的各项表决；获得执</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">行学代表职务所需的信息和各项保障；《四川大学学生会章程》规定的其他权利。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">学代表应当履行下列义务：模范地遵守法律、校纪校规；按时参加学生代表大会，认真审议各项提案、报告和其他议题；积极参加统一组织的视察、专题调研等履职活动；加强履职学习和调查研究，不断提高执行学代表职务的能力；与选举单位的同学保持密切联系，听取基层同学的意见和要求并向学生会反映；《四川大学学生会章程》规定的其他义务。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>三、学代表的构成要求</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">1.根据《四川大学学生会章程》及有关选举规定，从四川大学的实际情况出发，按照有利于组织和会议召开，有利于讨论和决定问题的原则，选区（学院）划分具体名额分配由&ldquo;学代会&rdquo;筹委会根据各学院学生人数、代表性、广泛性确定（附件1），四川大学第三十次学生代表大会拟设代表人数约占学生总人数10&permil;。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">2.2016级学代表占20％，2015级学代表占40％，2014级学代表占30％，2013、2012级（长学年制本科生）学代表共占10％；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">3.应充分保证学代表选举的广泛性，各选区女代表比例不低于30%，少数民族代表不低于10%。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">4.进一步扩大代表的广泛性，鼓励优秀的非学生会骨干同学成为学代表。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">5.学代表在具有广泛性的同时，应坚持先进性。适当优先考虑感动川大人物代表、德孝自强之星代表、网络知名人士代表、创业就业模范代表、高水平运动员代表等。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>四、学代表的产生</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">1.学代表候选人可向选区自荐或由选区推荐；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">2.各选区按照名额分配及构成要求进行选举，各选区通过召开学生代表大会或全体学生大会等方式，酝酿、选举学代表候选人，并于2017年4月28日（周五）前报送至学代会筹委会；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">3.各选区选举产生的学代表候选人，通过学代会筹委会学代表资格审查委员会审查、公示合格后获得正式代表资格。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>五、学代表材料汇总及提交</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">各选区选举产生学代表后，统一填写、报送学代表候选人信息表（附件2）、选区学代表候选人信息汇总表（附件3）。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">电子档于2017年4月28日22:00前打包发送至学代会筹委会邮箱：scuxdh@scuyouth.cn，纸质档以选区为单位于2017年4月28日12:00&mdash;18:00交至校学生会办公室（望江校区文华活动中心一楼）。</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;"><strong>六、注意事项</strong></span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">1.各选区在选举好学代表的过程中，要同时做好四川大学新一届学生会委员会候选人的酝酿、提名、考察工作，为选举产生四川大学新一届学生会委员会、主席团打好基础；</span></span></div> <div style="color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	<span style="font-family: 楷体_gb2312;"><span style="font-size: 20px;">2.四川大学新一届学生会委员会、主席团候选人须为学代会正式代表；</span></span></div> <div style="color: rgb(0, 0, 0);
