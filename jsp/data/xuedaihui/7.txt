<div class="content-art">  			<h1>关于开展2017-2018学年四川大学学生会主席团换届选举工作的通知</h1>  			<div class="attr">  				发布时间：2017-04-25 09:21:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/13836"></script>  			</div>  			<div class="content-text">  				<p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	各学院、全校同学：</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	为深入贯彻落实习近平总书记系列重要讲话和《学联学生会组织改革方案》等相关文件精神，强化四川大学学生会自我教育、自我管理、自我服务、自我监督的职能，激发动力、提升活力、增强吸引力和凝聚力，使学生会更好地代表和服务广大同学，更好地团结和凝聚广大同学听党话、跟党走，发挥&ldquo;一心双环&rdquo;在同学们成长成才中的重要作用，为推进&ldquo;四个全面&rdquo;战略布局、实现中华民族伟大复兴的中国梦贡献青春力量，拟定于2017年6月召开四川大学第三十次学生代表大会,大会将换届选举新一任四川大学学生会主席团。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	本次学生会换届选举工作坚持以&ldquo;公平、公正、公开&rdquo;为原则，选拔德才兼备的学生干部,以更好地为同学服务。现将具体事宜通知如下：</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	一、届次及设置</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	四川大学第三十届学生会委员会第一任主席团，即四川大学2017&mdash;2018学年学生会主席团。设学生会主席1名，副主席8名，其中3名副主席分别兼任校学生社团联合会主席、校学生学术科技协会主席、校青年志愿者协会主席。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	二、参选条件</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	1. 具有坚定的政治立场和政治方向，具备政治合格、学习优秀、品德良好、作风过硬、群众基础好在校期间无任何违纪违规行为；</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	2. 2017年5月至2018年6月在校就读（参加国际合作、对外交流等项目不得超过3个月）的全日制本科生，原则上面向2015级、2014级和长学年制专业的2013级本科生进行选拔；</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	3. 具有责任心和奉献精神，关注学校发展和同学权益，有一定的参政议政能力，具有良好的群众基础和影响力；</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	4. 专业成绩良好，无必修课重修记录，综合测评优秀；</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	5. 具有较丰富的服务广大同学的工作经验；</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	6. 同等条件下，中共党员优先，全国、省、市、校级大学生骨干培养学校优秀学员优先。适当考虑少数民族构成情况。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	三、选拔流程</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	推荐报名 &rarr; 学院审核 &rarr; 笔试 &rarr; 学生会工作论坛 &rarr; 拟入围面试同学基本信息及笔试成绩公示 &rarr; 面试 &rarr; 发布候选人通告&rarr;四川大学第三十次学生代表大会差额选举主席团</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	1. 推荐报名：符合条件的同学填写报名表（见附件），经四川大学学生会2016&mdash;2017学年主席团或校、院团委老师至少一人签字推荐后，方可提交学院审核。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	2. 学院审核：报名表须经所在班级、学院学生会、学院分团委、学院党委审核并签字盖章。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	3. 笔试：笔试分客观题、主观题。客观题涵盖党政方针、校史校情、学生会工作、共青团工作，主观题侧重工作思考、工作设想和个人能力考核。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	4. 学生会工作论坛：校、院学生会负责人，校团委老师，有意竞选学生会主席团、部门负责人的同学，关心学生会发展的同学参加，就学生会如何更好地发展、更好地服务同学进行交流探讨。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	5. 拟入围面试同学基本信息及笔试成绩公示：依据笔试排名按照一定比例拟定入围面试同学名单，并对笔试成绩、基本信息、学习成绩、任职经历等进行全校公示，接受师生意见。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	6. 面试：公示无异议的同学参加面试，邀请学校相关部处专家成立专家面试工作小组，对专业学习、工作设想、综合素质等方面进行评议。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	7. 发布候选人通告：依据专家面试工作小组的面试打分，结合工作表现、笔试成绩、年级、学科类别（文、理、工、医每类学科至少1人、每个学院最多1人）、性别、民族、交叉任职等因素，报送学校党委审批后，确定学生会主席团候选人名单，并进行全校通告。此环节不再进行公示、受理举报。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	8. 四川大学第三十次学生代表大会差额选举：主席团候选人在四川大学第三十次学生代表大会上进行竞选演讲，全体学生代表差额选举产生学生会主席团。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	四、报名及笔试</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	1. 所有符合参选条件的同学在完成相关推荐、审批后于4月21日(星期五)17:00前将报名表交至望江行政楼242办公室周若愚老师处；电子档报名表须于4月21日12:00前发送至邮箱：scuxdh@scuyouth.cn（报名邮箱已设置自动回复，请注意核对）。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	2. 笔试：拟4月下旬以闭卷形式进行。</p> <p style="margin: 25px 0px; padding: 0px; text-indent: 2em; font-size: 15px; line-height: 1.5em; color: rgb(0, 0, 0); font-family: &quot;Microsoft YaHei&quot;, Arial, sans-serif; background-color: rgb(236, 240, 242);"> 	五、注意事项</p> <p style="margin: 25px 0px;
