<div class="content-art">  			<h1>四川大学校级学生组织印刷品制作管理制度</h1>  			<div class="attr">  				发布时间：2014-09-12 18:17:00  浏览次数：<script src="/index.php/system/web/view/viewnumber/id/5496" type="text/javascript"></script>1453  			</div>  			<div class="content-text">  				<div>  	&nbsp;</div>  <h2>  	<span style="font-family:宋体;">第一条</span> <span style="font-family:宋体;">为进一步贯彻落实党的十八大提出的“艰苦奋斗、勤俭节约”精神，规范校团委直属或指导的校级学生组织印刷品制作，结合我校各学生组织的工作现状，特制定本制度。</span></h2>  <h2>  	<span style="font-family:宋体;">第二条</span> <span style="font-family:宋体;">本制度由校团委秘书处负责监督实行，适用于由校团委直属或指导的校级学生组织。</span></h2>  <h2>  	<span style="font-family:宋体;">第三条</span> <span style="font-family:宋体;">可制作的印刷品类别主要包括黑白打复印、彩印、海报印制三类，以及座牌、信函等。</span></h2>  <h2>  	<span style="font-family:宋体;">第四条</span> <span style="font-family:宋体;">印刷品管理原则</span></h2>  <div>  	<span style="font-family:宋体;"><span style="font-size:14.0pt;">（一）印刷品的内容不得违反国家、学校的相关规定。</span></span></div>  <div>  	<span style="font-family:宋体;"><span style="font-size:14.0pt;">（二）各类印刷品的制作都要秉持“厉行节约、反对浪费”的原则，一般应遵循如下规定：</span></span></div>  <div>  	<span style="font-size:14.0pt;">1</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、邀请函、会议记录、通讯录等能够以电子文档形式存档或交流的文件应尽量使用电子文档。</span></span></div>  <div>  	<span style="font-size:14.0pt;">2</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、单次常规活动的展板张贴海报总面积不得超过</span></span><span style="font-size:14.0pt;">4m<sup>2</sup></span><span style="font-family:宋体;"><span style="font-size:14.0pt;">（约三张海报），特殊情况需单独提出申请。</span></span></div>  <div>  	<span style="font-size:14.0pt;">3</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、原则上所有黑白打印及复印的材料都在江安行政楼</span></span><span style="font-size:14.0pt;">407</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">校团委办公室和望江校团委办公室进行印制，特殊情况需单独提出申请。原则上单次活动复印</span></span><span style="font-size:14.0pt;">A4</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">大小宣传单不超过</span></span><span style="font-size:14.0pt;">200</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">页。</span></span></div>  <div>  	<span style="font-size:14.0pt;">4</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、如无特殊情况，原则上各学生组织例会均不打印、复印会议资料，尽量使用电子文档。</span></span></div>  <div>  	<span style="font-size:14.0pt;">5</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、各类活动如无特殊情况，均不在围合或宿舍张贴彩印海报。</span></span></div>  <div>  	<span style="font-size:14.0pt;">6</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、各类活动如无特殊情况，均不制作横幅。</span></span></div>  <h2>  	<span style="font-family:宋体;">第五条</span> <span style="font-family:宋体;">印刷品制作必须在校团委规定的地点进行。</span></h2>  <div>  	<strong><span style="font-family:宋体;"><span style="font-size:14.0pt;">黑白打印、复印：</span></span></strong><span style="font-family:宋体;"><span style="font-size:14.0pt;">校团委江安<span style="color:black;">行政楼</span></span></span><span style="color:black;"><span style="font-size:14.0pt;">407</span></span><span style="color:black;"><span style="font-family:宋体;"><span style="font-size:14.0pt;">校团委办公室、望江校区行政楼校</span></span></span><span style="color:black;"><span style="font-size:14.0pt;">240</span></span><span style="color:black;"><span style="font-family:宋体;"><span style="font-size:14.0pt;">团委办公室</span></span></span></div>  <div>  	<strong><span style="color:black;"><span style="font-family:宋体;"><span style="font-size:14.0pt;">彩印：</span></span></span></strong><span style="font-family:宋体;"><span style="font-size:14.0pt;">江安</span></span>&nbsp; <span style="font-family:宋体;"><span style="font-size:14.0pt;">永真打印社</span></span><span style="font-size:14.0pt;">&nbsp;&nbsp;&nbsp; 85990050&nbsp;&nbsp;&nbsp;&nbsp; </span></div>  <div>  	&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<span style="font-family:宋体;"><span style="font-size:14.0pt;">望江</span></span>&nbsp; <span style="font-family:宋体;"><span style="font-size:14.0pt;">涉水人打印社</span></span><span style="font-size:14.0pt;">&nbsp; 85259788/13688358517</span></div>  <h2>  	<span style="font-family:宋体;">第六条</span> <span style="font-family:宋体;">印制流程（流程图见附件</span>1<span style="font-family:宋体;">）</span></h2>  <div>  	<span style="font-family:宋体;"><span style="font-size:14.0pt;">（一）校团委江安办公室（江安行政楼</span></span><span style="font-size:14.0pt;">407</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">室）</span></span></div>  <div>  	<span style="font-family:宋体;"><span style="font-size:14.0pt;">工作人员持印刷工作证在办公室校团委秘书处值班同学处填写《四川大学校团委江安办公室打、复印登记簿》（附件</span></span><span style="font-size:14.0pt;">3</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">）后方可打印、复印，工作时间为每周一至周五上午</span></span><span style="font-size:14.0pt;">9:00-12:00</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">，下午</span></span><span style="font-size:14.0pt;">2:30-5:30</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">。</span></span></div>  <div>  	<span style="font-family:宋体;"><span style="font-size:14.0pt;">（二）江安：永真打印社、望江：涉水人打印社</span></span></div>  <div>  	<span style="font-size:14.0pt;">1</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、定价：申请人根据需制作印刷品的类别、尺寸、数量向印刷店确定该印刷品的印制价格。</span></span></div>  <div>  	<span style="font-size:14.0pt;">2</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、申请与审核：申请人下载填写《四川大学校级学生组织印刷品制作申请表》（附件</span></span><span style="font-size:14.0pt;">2</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">），交由该学生组织所在指导老师签字，审批通过的《四川大学校级学生组织印刷品制作申请表》第三联由学生组织自行保管留底以便日后审查。</span></span></div>  <div>  	<span style="font-size:14.0pt;">3</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、将审批通过的《四川大学校级学生组织印刷品制作申请表》第二联交到校团委办公室（</span></span><span style="font-family:宋体;"><span style="font-size:14.0pt;">校团委江安办公室（江安行政楼407</span>室）或校团委望江办公室（望江行政楼240</span>室））<span style="font-family:宋体;"><span style="font-size:14.0pt;">秘书处值班人员处，并填写《四川大学团委印刷品制作登记簿》。</span></span></div>  <div>  	<span style="font-size:14.0pt;">4</span><span style="font-family:宋体;"><span style="font-size:14.0pt;">、印制：申请人持印制工作证、审核通过的《四川大学校级学生组织印刷品制作申请表》第一联于印刷店进行打印，完成印制后再依照所属学生组织填写店内《四川大学校级学生组织印刷品制作签单簿》。</span></span></div>  <h2>  	<span style="font-family:宋体;">第七条</span> <span style="font-family:宋体;">校团委秘书处每月月底将对本月各学生组织在打印店内的《四川大学校级学生组织印刷品制作签单簿》与校团委办公室的《四川大学团委印刷品制作登记簿》进行对账审核</span>,<span style="font-family:宋体;">进行统计后，于团委办公会上进行公布。若有疑问，申请人须在</span>5<span style="font-family:宋体;">个工作日内详细说明问题原因与解决方案。</span></h2>  <p>  	<span style="font-size:16px;"><strong><span style="font-family:宋体;">附件1：<a
