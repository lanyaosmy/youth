<div class="content-art">  			<h1>共青团成都市委关于在青少年中大力开展社会主义荣辱观学习实践活动的通知 成青发〔2006〕9号</h1>  			<div class="attr">  				发布时间：0000-00-00 00:00:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/3532"></script>  			</div>  			<div class="content-text">  				<div align="center"> &nbsp;</div> <div align="left"> <span style="font-size:16px;">共青团各区（市）县委，各直属团工委，市属各机关、企事业单位团委（总支、支部），各大学、中专、中学团委：</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 3月4日，胡锦涛总书记在看望出席全国政协十届四次会议的委员时强调指出，要引导广大干部群众特别是青少年树立社会主义荣辱观，坚持以热爱祖国为荣、以危害祖国为耻，以服务人民为荣、以背离人民为耻，以崇尚科学为荣、以愚昧无知为耻，以辛勤劳动为荣、以好逸恶劳为耻，以团结互助为荣、以损人利己为耻，以诚实守信为荣、以见利忘义为耻，以遵纪守法为荣、以违法乱纪为耻，以艰苦奋斗为荣、以骄奢淫逸为耻。胡锦涛总书记的重要论述，概括精辟，寓意深刻，代表了先进文化的前进方向，体现了社会主义基本道德规范的本质要求，体现了依法治国与以德治国相结合的治国方略，是我们党关于社会主义道德建设思想的继承和发展，是进一步推进精神文明建设的重要指导方针。全市各级团组织要深刻领会胡锦涛总书记的重要论述，把在青少年中大力开展社会主义荣辱观学习实践活动作为全市青少年思想道德建设的重要内容，作为当前和今后一个时期的重要工作，切实抓紧抓实抓好。现将有关要求通知如下。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 1、要充分认识树立社会主义荣辱观对当代青少年的重要意义。胡锦涛总书记在讲话中特别强调，要引导广大干部群众特别是青少年树立社会主义荣辱观。这既充分体现了以胡锦涛同志为总书记的党中央对青少年一代的亲切关怀和殷切期望，又指明了青少年树立社会主义荣辱观的重要性。青少年处在生长发育的关键时期，树立正确的价值观念，帮助他们明辨是非、美丑、善恶，对他们的成长至关重要。特别是在当前社会转型时期，面对社会思想道德建设中出现的种种复杂情况，加强对青少年的社会主义荣辱观教育具有重要的现实意义。全市各级团组织要充分认识树立社会主义荣辱观对当代青少年的重要意义，针对青少年特点，运用青少年喜闻乐见、易于接受的方式，帮助青少年理解掌握社会主义荣辱观的时代背景、重要意义和丰富内涵，增强牢固树立社会主义荣辱观的自觉性。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 2、要帮助广大青少年不断加深对社会主义荣辱观的认识理解。胡锦涛总书记关于社会主义荣辱观的重要论述，涵盖爱国主义、集体主义、社会主义思想，以及中华民族传统文化和道德规范的要求，是社会主义价值观、道德观的具体体现，是科学的世界观、人生观、价值观的有机统一，为全社会确立了看得见、摸得着的道德规范和行为准则，具有鲜明的时代特征，为共青团进一步做好新时期的青少年工作指明了方向。全市各级团组织一定要紧密结合工作实际，认真学习、深刻领会胡锦涛总书记的重要论述，按照&ldquo;八荣八耻&rdquo;的基本要求，以&ldquo;成都和谐发展，青年创业成才&rdquo;为总揽，在青少年中广泛深入地大力宣传和倡导社会主义荣辱观，为推进我市统筹城乡、四位一体科学发展提供精神动力和智力支持。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 3、要在青年职工中深入开展社会主义荣辱观学习实践活动。紧紧围绕市委&ldquo;加快产业发展年&rdquo;确定的目标任务，把青年职工树立社会主义荣辱观落实到推进我市产业结构优化升级的具体实践中。要在青年职工中广泛开展&ldquo;三创一兴&rdquo;、青年文明号、青年突击队等活动，增强青年职工的光荣感、责任感和使命感，引导青年职工创新创造、岗位成才，弘扬职业文明，激励青年职工发愤学习新知识新技能，努力掌握核心技术和关键技术，提升产业技术水平，在调整产业结构、转变经济增长方式中发挥生力军作用。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 4、要在农村青年中大力开展社会主义荣辱观学习实践活动。要围绕市委建设社会主义新农村的总体安排，逐步使&ldquo;八荣八耻&rdquo;的基本要求融入广大农村青年的学习、工作和生活之中，组织农村青年在丰富生动的社会实践中学习宣传社会主义荣辱观。要根据农村实际，结合&ldquo;三下乡&rdquo;、农村青年就业创业、保护母亲河行动、乡村青年文化节等活动，因地制宜，积极创新，运用有效的手段和方法，引导他们从日常行为做起，从我做起，从小事做起，从现在做起，破除腐朽落后思想影响，追求文明健康生活方式，勇开风气之先，弘扬时代新风正气，促进农村青年综合素质的提高，促进农村良好道德风尚的形成，为建设社会主义新农村提供强大的精神动力和思想保证。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 5、要在青少年学生广泛开展社会主义荣辱观学习实践活动。各学校团组织要利用网络、校园电视台、广播站、板报、墙报、宣传栏等阵地，大力宣传社会主义荣辱观的丰富内涵，宣传青少年中树立社会主义荣辱观的先进典型，宣传开展社会主义荣辱观教育的好经验、好做法，形成良好导向，营造浓厚氛围。要通过升旗仪式、入团仪式、团课、班会、团队活动、演讲、座谈等形式，组织广大青少年学生学习&ldquo;八荣八耻&rdquo;，并开展&ldquo;树立正确荣辱观&rdquo;大讨论。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 6、要使社会主义荣辱观在广大青少年中深入人心。全市各级团组织要充分发挥团属舆论阵地在宣传社会主义荣辱观中的重要作用，认真安排部署，加强整体策划，做好重点宣传，在广大青少年中和全社会宣传普及社会主义荣辱观。要大力宣传社会主义荣辱观的丰富内涵，形成良好导向，营造浓厚氛围。团属青少年宫、青少年教育基地、青年中心等青少年活动阵地要发挥自身优势，发掘教育资源，针对青少年特点，运用青少年喜闻乐见、易于接受的方式，积极宣传社会主义荣辱观，大力倡导&ldquo;八荣八耻&rdquo;，让社会主义荣辱观深入青少年和广大群众心中。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 7、要把社会主义荣辱观学习实践活动与巩固团员意识教育活动成果相结合。各级团组织要继续开展&ldquo;举团旗、戴团徵、学团章、交团费、唱团歌、过团日&rdquo;主题活动，围绕新时期共青团员的时代特征、素质能力、作用发挥等专题展开讨论，切实解决当前团员队伍中存在的突出问题，达到提高团员素质、巩固团组织建设、促进团的各项工作的根本目的，切实巩固团员意识教育活动的成果。要把树立社会主义荣辱观作为团员意识教育活动的重要内容，切实增强政治意识，提高团员对团组织的认同感和归属感。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 8．各级团干部要在社会主义荣辱观学习实践活动中发挥榜样带动作用。广大团干部要带头深入学习社会主义荣辱观，切实掌握精神实质，努力把握基本要求。要按照&ldquo;八荣八耻&rdquo;的要求，规范思想行为，加强道德修养，培养高尚情操。要加强和改进团的作风建设，深入基层、深入实际、深入青年，创造性地开展工作，增强工作的针对性、实效性。要勤于学习、善于创造、甘于奉献，始终忠诚党的事业、热爱团的岗位、竭诚服务青年，努力做&ldquo;党放心、青年满意&rdquo;的团干部。</span></div> <div align="left"> <span style="font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; 团市委号召，全市各级团组织要切实做好宣传发动工作，迅速掀起开展社会主义荣辱观学习实践活动的热潮，把广大团员青年的智慧和力量凝聚到实现&ldquo;三最&rdquo;宏伟目标的伟大实践中。各基层团组织要及时总结活动成果，有关活动开展情况请及时上传&ldquo;成都共青团网站&rdquo;（WWW.CDCYL.COM.CN）,报团市委宣传与研究部。</span></div> <div align="left"> &nbsp;</div> <div align="left"> &nbsp;</div> <div align="right"> <span style="font-size:16px;">共青团成都市委</span></div> <div align="right"> <span style="font-size:16px;">2006年3月20日</span></div> <div align="left"> &nbsp;</div>  			</div>  		</div>
