<div class="content-art">  			<h1>关于组织学生参与第四届“创青春”四川青年创新创业大赛暨第八届高校毕业生创业大赛的通知</h1>  			<div class="attr">  				发布时间：2017-08-02 21:41:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14179"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family: 楷体_gb2312; font-size: 18px;">全校同学：</span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;</span></span><span style="font-family: 楷体_gb2312; font-size: 18px;">为鼓励和培养川大学子的创业精神，锻炼和提高川大学子的创新能力和实践能力，</span><span style="font-family: 楷体_gb2312; font-size: 18px; color: rgb(51, 51, 51);">我校将组织广大同学参加</span><span style="font-family: 楷体_gb2312; font-size: 18px; color: rgb(14, 14, 14);">第四届&ldquo;</span><span style="font-family: 楷体_gb2312; font-size: 18px;">创青春&rdquo;四川青年创新创业大赛暨第八届高校毕业生创业大赛。四川省高校毕业生创业大赛自2009年创办以来已经成功举办了七届，赢得了全省青年学生的广泛参与和社会各界的好评。我校连续七年夺得奖牌，在2016年举办的第三届&ldquo;创青春&rdquo;四川青年创新创业大赛暨第七届高校毕业生创业大赛中，我校荣获了8金、11银、10铜的好成绩。现将相关事宜通知如下：</span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>一、大赛主题</strong></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><span style="color:#0e0e0e;">&nbsp; &nbsp; &nbsp; &nbsp;创新引领未来&middot;创业成就梦想</span></span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>二、参赛形式</strong></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;以个人或团队形式参赛，每个团队人数不超过5人。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>三、参赛对象</strong></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（1）个人形式参赛对象</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;在校生和毕业 5 年以内的四川大学毕业生，即可以个人名义申报参赛项目。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（2）团队形式参赛对象</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;凡团队申报的项目，团队总人数不多于 5 人，团队主要负责人年龄不超过 35 周岁，其他成员年龄应不超过 40 周岁，且团队平均年龄不超过 35 周岁，要求团队成员均为四川大学在校学生或毕业生。</span></span></div> <div> 	&nbsp;</div> <div> 	<strong style="font-family: 楷体_gb2312; font-size: 18px;">四、参赛项目要求</strong></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（1）符合国家法律法规和国家产业政策。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（2）不得侵犯他人知识产权。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（3）具有良好的经济效益、社会效益，经营规范，社会信誉良好。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（4）尚未接受过投资或仅接受过早期投资（种子轮、天使轮或A 轮投资）。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（5）掌握具有较大投资价值的独特产品、技术或商业模式。</span></span></div> <div> 	&nbsp;</div> <div> 	<strong style="font-family: 楷体_gb2312; font-size: 18px;">五、大赛赛制</strong></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;大赛分为主体赛事（商工组、现代农业和农村电子商务组）和互联网组赛事。另设 APP 专项赛。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（1）商工组重点关注节能环保、新一代信息技术、生物、高端装备制造、新能<span style="color:#0e0e0e;">源、新材料、新能源汽车等相关产业。</span></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><span style="color:#0e0e0e;">（2）现代农业和农村电子商务组重点关注种养殖、农产品加工、农业社会化服务及农村电子商务等相关产业。对带动建档立卡贫困户脱贫的项目给予重点倾斜。</span></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><span style="color:#0e0e0e;">（3）互联网组重点关注互联网科技创新、互联网基础服务、应用服务及运用互联网手段改造发展传统行业等相关产业。</span></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><span style="color:#0e0e0e;">（4）APP 专项赛重点关注移动互联网技术及应用开发项目.</span></span></span></div> <div> 	&nbsp;</div> <div> 	<strong style="font-family: 楷体_gb2312; font-size: 18px;">六、作品分组规则</strong></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;根据参赛项目所处的创业阶段及企业创办年限（以工商登记为准）不同，分设组别。其中，商工组赛设初创组、成长组；现代农业和农村电子商务组赛设初创组、成长组；互联网组赛、APP专项赛设创意组、创业（应用）组。企业创办年限以 2017 年 6月 30 日为界限进行划分。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（1）创意组申报规则。产品原型或服务模式已研发成型，具有成熟的商业计划书，但是尚未完成工商登记注册的创业项目。申报参赛须提交商业计划书、市场调研报告，有清晰的发展模式和可靠的数据支持。第一申报人须为项目主要负责人。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（2）初创组申报规则。工商登记注册时间不超过 2 年（含）的企业创业项目。申报参赛须提交营业执照、税务登记证副本、银行开户许可证复印件等相关文件，特色产品认定证书等相关证件，项目成长过程或生产流程相关介绍，项目发展构想及阶段性成果等资料。第一申报人须为企业法人代表，且所占股份不低于 30%（含）。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（3）成长组申报规则。工商登记注册时间在 2 至 5 年（含）之间的企业创业项目。申报参赛须提交营业执照、税务登记证副本、银行开户许可证复印件等相关文件，特色产品认定证书等相关证件，项目成长过程或生产流程相关介绍，项目发展构想及阶段性成果等资料。第一申报人须为企业法人代表，且所占股份不低于30%（含）。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（4）互联网组及 APP 专项赛创业（应用）组申报规则。工商登记注册时间在 5 年 (含) 以内的创业项目。申报参赛须提交营业执照、税务登记证副本、银行开户许可证复印件、项目计划书等相关文件；APP 专项赛创业组还应提供应用程序(必须是基于安卓或 IOS 系统设计开发的 APP 应用程序)。参赛团队应对项目拥有所有权和知识产权。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>七、赛程安排</strong></span></span></div> <div> 	<span style="font-family: 楷体_gb2312; font-size: 18px;">2017年8月4日</strong><span style="font-family: 楷体_gb2312; font-size: 18px;">。</span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（2）初赛。2017 年 8月上旬将开展大赛初赛，组委会将组织评审委员会对报名项目进行评审。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（3）决赛。暂定于 2017 年 8 月在南充市举办大赛决赛。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>八、奖励及激励</strong></span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp;</span></span><span style="font-family: 楷体_gb2312;
