src="/index.php/system/web/view/viewnumber/id/14135"></script>  			</div>  			<div class="content-text">  				<div align="center"> 	&nbsp;</div> <div> 	<span style="font-family: 宋体"><span style="font-size: 22px">各学院团委、全校同学：</span></span></div> <div> 	<span style="font-family: 宋体"><span style="font-size: 22px">为深入学习宣传贯彻习近平总书记系列重要讲话和五四重要讲话精神，组织引领广大青年学生深入社会、了解国情、接受锻炼，以优异成绩喜迎党的十九大胜利召开；为鼓励大学生积极参加社会调研活动、创新创业活动、单位实习实训活动，让大学生经受锻炼、提高能力，同时也为大学生毕业后的就业创造条件、提供机遇，经与湖北省黄冈市沟通，组织开展2017年暑期&ldquo;三下乡&rdquo;赴黄冈市社会实践活动。现将有关事项通知如下。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体"><strong>一、时间地点</strong></span></span></div> <p> 	<span style="font-family: 宋体"><span style="font-size: 22px"><strong><span 	<span style="font-size: 22px"><span style="font-family: 宋体"><strong>(注：根据黄冈市岗位要求，实习实训时长不得少于半个月)</strong></span></span></p> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体"><strong>二、对象</strong></span></span></div> <p> 	<span style="font-size: 22px"><span style="font-family: 宋体"><span style="mso-ascii-theme-font: minor-fareast; mso-fareast-theme-font: minor-fareast; mso-hansi-theme-font: minor-fareast; mso-bidi-font-family: &quot;Times New Roman&quot;; mso-bidi-theme-font: minor-bidi; mso-ansi-language: EN-US; mso-fareast-language: ZH-CN; mso-bidi-language: AR-SA"><font color="#000000">全体在校学生，鼓励就近就便参与。</font></span></span></span></p> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体"><strong>三、活动安排</strong></span></span></div> <div align="left"> 	<span style="font-size: 22px"><span style="font-family: 宋体">请有参与意愿的同学依照实践专项项目进行申报，此次社会实践项目主要包括：</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">1.红色教育专项：限学院组队报名。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">2.创新创业专项：限学院组队报名。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">3.基层调研专项：限学院组队报名。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">4.实习实训专项：可自行组队申报，实训单位为黄冈市机关或企事业单位。</span></span></div> <div> 	<span 	<span style="font-size: 22px"><span style="font-family: 宋体">同学们进行学院重点组队组成调研团队，由学校按照公平、公正、公开的原则进行选拔，周边地区同学优先。调研团队自行选择项目，<strong>填写附件二、附件四、附件五</strong>，将电子版于<span style="color: red">7</span><span red">月7</span>日9：00&mdash;12:00交送至望江行政楼团委242办公室。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">2.团队遴选</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">学校将根据申报情况遴选确定若干支调研团队进行立项。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">3.参加活动</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">校团委将通过邮件和电话形式通知团队入选情况及参加活动具体事项，各入选团队按实践计划安排赴湖北省黄冈市参加社会实践活动。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">4.经费支持与实践团队评定</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">主办单位将提供其它具体需求的支持和帮助。实践活动结束后将组织有关专家对调研团队和课题成果进行评审，遴选出优秀实践团队、优秀调研成果进行表彰奖励。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体"><strong>五、实习实训项目须知</strong></span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">1.务必在完成校内申报、完成在黄冈市大学生实习实训服务信息平台完成报名确认后再前往!</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">2.有意到黄冈市直实习实训的大学生,必须在黄冈市大学生实习实训服务信息平台报名（dxs.hg12333.com）。黄冈市人才服务局每月25号前对下个月的报名信息进行审核、确认，并在网上公告（学院组队报名除外）。学生原则上要求每月10日前到黄冈市人才服务局办理实习实训报到上岗手续。凡自行联系实习单位，没有及时在黄冈市人才服务局办理报名登记手续的，不享受黄冈市直大学生实习实训有关政策待遇。每年元月、2月、12月一般不安排实习生实习。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">3.实习学生携带学生证和身份证到黄冈市人才服务局领取《实习实训报到通知单》和《实习实训后勤基地报到通知单》，凭《实习实训报到通知单》到实习单位报到；凭《实习实训后勤基地报到通知单》到实习实训后勤基地领取饭卡、办理食宿手续，并签订《黄冈市大学生实习实训承诺书》。报道通知单有效期为五天，逾期作废。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">4.实习期间必须严格遵守实习单位、后勤保障基地各项管理规定。如需要请假的，须书面提出申请，注明事由、起止时间，两天以内的（含两天），由实习单位批准，并将请假条复印件报后勤基地寝室管理员备案后方可离开；超过两天的，须由黄冈市人才服务局批准后方可离开。事假期满，须按时返回销假。不在后勤基地住宿的，须经后勤保障基地同意，安全自负。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">5.实习时间一般为一至三个月，实习期满一个月需要继续实习的大学生，须填写《大学生实习实训延长实习时间申请表》（表格在实习实训信息平台下载中心下载），经黄冈市人才服务局签署批准意见后方可办理饭卡充值手续。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">6.大学生实习结束，填写《大学生实习实训鉴定表》一式三份（实习单位、人才服务局、所在高校各一份）和《实习结束告知书》一份（表格可从实习实训网站下载）。实习大学生凭《大学生实习实训鉴定表》和《实习结束告知书》到后勤基地办理实习结束手续，清退饭卡余额。在后勤基地食宿的学生，须退还生活用品等相关物品。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family: 宋体">7.实习实训大学生事业单位专项招聘事宜，请关注黄冈市人力资源和社会保障局网站（<a href="http://www.hg12333.com">www.hg12333.com</a>）和黄冈市大学生实习实训服务信息平台报名（dxs.hg12333.com）。</span></span></div> <div> 	<span style="font-size: 22px"><span style="font-family:
