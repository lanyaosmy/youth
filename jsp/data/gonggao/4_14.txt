<div class="content-art">  			<h1>关于组织申报2017年度 大学生“小平科技创新团队”的通知</h1>  			<div class="attr">  				发布时间：2017-06-21 16:36:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14120"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">各学院、全体同学：</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 中国青少年科技创新奖励基金是在2004年邓小平同志百年诞辰之际，根据小平同志的遗愿，小平同志亲属捐献出小平同志生前全部稿费，委托共青团中央、全国青联、全国学联、全国少工委共同设立的。基金用以表彰&ldquo;中国青少年科技创新奖&rdquo;获得者和选拔&ldquo;小平科技创新团队&rdquo;予以科技创新研发基金支持。我校制造科学与工程学院2012级硕士研究生曾杰等同学组建的&ldquo;四川大学管道检测项目研发团队&rdquo;曾于2014年获评&ldquo;小平科技创新团队&rdquo;，并在人民大会堂接受中共中央政治局委员、国务院副总理刘延东同志的颁奖。</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 为进一步加强大学生科技创新能力的培养，扩大中国青少年科技创新奖励基金（基金简介见附件1）的社会影响，中国青少年科技创新奖励基金拟于2017年支持50个大学生科技创新团队开展科技创新研究，并拟命名为大学生&ldquo;小平科技创新团队&rdquo;。请符合要求的单位积极组织申报，具体要求如下：</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 一、申报对象</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 在学术研究、科技竞赛、成果转化等方面取得突出成绩或显示较大潜力的大学生科技创新团队（学生社团），原则上每个创新团队（学生社团）不少于10人。申报的团队应具有以下基本条件：</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 1. 有明确的学术领域，专注于特定学科或特定课题的学术前沿领域，团队成立时间不少于5年；</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 2. 有规范的组织形态，以兴趣团队、研究团队等体现，具有跨年级的人员梯队培养机制、跨专业的人员交流机制，以及规范的组织运作机制等；</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 3. 有一定的阶段成果，如在&ldquo;挑战杯&rdquo;、&ldquo;创青春&rdquo;等科技竞赛中获奖、发表高水平学术论文或产生良好的创新创业示范效应等；</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 4. 有固定的支持体系，应具备固定的活动场所、稳定的指导教师团队以及研究经费等。</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 二、经费资助额度</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 入选的每个团队资助经费4万元，全国范围内共支持50个团队，由中国青少年科技创新奖励基金支持。</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 三、申报工作安排</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp; &nbsp; 各申报团队需准备一个8分钟的视频，并于2017年6月28日15:00-17:00将《大学生&ldquo;小平科技创新团队&rdquo;申报表》（申报表见附件2）一式三份提交至校团委（望江行政楼办公室240），同时将电子版报名表和8分钟视频发送至scuxsxskjxh@163.com（邮件标题请注明&ldquo;队长姓名+院系+联系电话+&lsquo;小平科技创新团队&rsquo;&rdquo;字样）。校团委将组织相关专家综合评议、择优推荐，原则上2016年已获得大学生&ldquo;小平科技创新团队&rdquo;称号的团队不再推荐。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">联系方式：</span></span></div> <div> 	<span style="font-family:楷体_gb2312;"><span style="font-size:18px;">&nbsp;
