src="/index.php/system/web/view/viewnumber/id/14174"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">全校各级团组织、团员青年：</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　为认真学习贯彻中国共产党四川大学第八次代表大会精神，<span style="color:black;">引导广大青年学生以更加奋发有为的精神状态，凝聚起全面推进学校&ldquo;两个伟大&rdquo;的强大合力，</span>为学校各项事业发展提供坚强保证，校团委决定在全校范围内开展认真学习贯彻学校第八次党代会精神活动，现就有关工作事项通知如下。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:#444444;">　　一、统一思想，深刻认识认真学习贯彻学校第八次党代会精神的重大意义</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　中国共产党四川大学第八次代表大会<span style="color:black;">是在全党全国迎接党的十九大召开之际、学校全面推进世界一流大学建设关键时期召开的一次重要会议</span>。大会高举中国特色社会主义伟大旗帜，深入学习贯彻习近平总书记系列重要讲话精神和治国理政新理念新思想新战略，回顾总结了学校第七次党代会以来的工作，科学谋划了学校党的建设新的伟大工程和建设世界一流大学新的伟大事业的奋斗目标、主要任务和重要举措，选举产生了新一届中共四川大学委员会和纪律检查委员会，团结带领全校党员干部和师生员工为建设具有中国特色、川大风格的世界一流大学而努力奋斗。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　认真学习贯彻学校第八次党代会精神，关系学校工作全局和长远发展，对动员全校团员青年在学校党委领导下，全面推进学校党的建设新的伟大工程和建设世界一流大学新的伟大事业具有重大现实意义和深远历史意义。全校各级团组织、团员青年要充分认识认真学习贯彻学校第八次党代会精神的重要意义，把思想和行动统一到会议精神上来，切实增强学习贯彻的思想自觉和行动自觉，不忘初心、砥砺奋进，努力开创学校事业发展新局面。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:#444444;">　　二、</span>深刻领会，准确把握学校第八次党代会的精神实质</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　各级团组织要以认真学习贯彻学校第八次党代会精神为契机扎实做好共青团思想政治工作，紧紧围绕立德树人这一中心环节,将学生思想政治工作放在共青团工作的突出位置,在全程育人、全方位育人中强化责任担当，重点把握以下几个方面。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　（一）深刻领会学校<span style="color:black;">第七次党代会以来的主要成绩和基本经验。</span>过去六年学校在<span style="color:black;">人才培养、师资队伍建设、科学研究、文化育人、社会服务、对外交流合作、现代大学制度建设、和谐校园建设、党的建设等方面取得</span>了重大成就。同时总结出坚持党的领导，全面从严治党；坚持立德树人，把牢正确办学方向；坚持改革创新，聚焦质量核心，增强发展活力；坚持以人为本，全心全意依靠教职工办学，走人才强校之路；坚持川大风格，走特色发展之路的基本经验。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　（二）深刻领会<span style="color:black;">学校党的建设新的伟大工程和建设世界一流大学新的伟大事业的宏伟目标</span>。站在历史新起点上，从世界变革、国家发展、民族振兴的大格局中明确大会报告中提出的&ldquo;力争到建党100周年、学校建校125周年，学校核心指标师均量进入全国高校前列，人才培养质量全国领先，科学研究、社会服务和文化传承创新水平位居国际前列，若干学科进入世界一流行列。力争到2030年、学校建校135周年左右，更多学科进入世界一流行列，学校的国际知名度和影响力显著提升，对国家、区域经济社会发展的支撑作用显著增强，率先走出一条西部高校建设世界一流大学的特色发展道路。力争到建国100周年、学校建校150周年左右，一批学科进入世界一流行列，若干学科进入世界一流前列，学校综合实力进入全球高校前150强，成为培养具有全球竞争力人才、胜任未来发展挑战的大学；成为协同引领战略新兴产业关键核心技术、提供经济社会发展高端智库智力支持的大学；成为产生新思想、新知识、新理论、新技术、新方法，促进世界和人类进步的大学；成为引领社会先进文化发展方向，守护人类精神家园和社会文明灯塔的大学，基本建成具有中国特色、川大风格的世界一流大学&rdquo;的奋斗目标。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　（三）深刻领会<span style="color:black;">学校今后五年工作的总体要求和主要任务。坚持高举中国特色社会主义伟大旗帜，深入学习贯彻习近平总书记系列重要讲话精神和治国理政新理念新思想新战略，紧紧围绕全面推进&ldquo;两个伟大&rdquo;这条主线，加强党的领导，坚持以一流为目标，以学科为基础，以绩效为杠杆，以改革为动力，全面加快世界一流大学建设步伐。为了实现四川大学的宏伟目标，近五年内学校要切实抓好以下工作：坚持立德树人，培养具有国际竞争力的一流拔尖创新人才；深入实施人才强校战略，建设一流师资队伍；全面加强学科和科研建设，打造一流学科体系和创造一流科研成果；传承创新优秀文化，繁荣发展哲学社会科学；充分发挥综合性大学优势，提供一流社会服务；实施高端国际合作发展战略，全面提升国际影响力和竞争力；完善学校治理体系，推进一流大学治理能力现代化；持续改善民生，建设一流校园环境；坚定不移推进全面从严治党，引领保障世界一流大学奋力前行。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:#444444;">　　三、扎实推进，确保认真学习贯彻学校第八次党代会精神取得实效</span></span></span></div> <div> 	<span 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　二要深入宣传，营造浓厚氛围。针对团员多样化学习需求，充分利用网站、微博、微信等新媒体平台，开发制作形象直观、丰富多样的学习资源，及时推送学习教育内容，在校内营造浓厚的学习氛围。</span></span></div> <div> 	<span
