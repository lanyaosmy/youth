<div class="content-art">  			<h1>关于组织开展2017年四川大学“欧莱雅助学金”评选工作的通知</h1>  			<div class="attr">  				发布时间：2017-10-13 11:04:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14514"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-size:16px;"><span style="font-family:宋体;">各学院团委：</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">&nbsp; &ldquo;欧莱雅&rdquo;校园公益项目已在全国范围内持续开展十余年，通过助学金评选工作、校园公益义卖工作、校园公益服务立项工作促进青年大学生成长成才，践行社会主义核心价值观。&ldquo;欧莱雅助学金&rdquo;评选旨在为品学兼优、家庭经济困难的优秀学生缓解经济压力，搭建能力、素质提升的平台，创造推外交流的机会，四川大学团委联合中国青少年发展基金会、欧莱雅（中国）有限公司，拟于近期开展四川大学2017年&ldquo;欧莱雅助学金&rdquo;评选，资助经费来源于2016年四川大学欧莱雅校园公益义卖。现将有关事项通知如下：</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:20px;"><strong><span style="font-family:宋体;">一、资助对象：</span></strong></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">四川大学全日制在校本科生及研究生</span></span></div> <div> 	&nbsp;</div> <div> 	<strong><span style="font-size:20px;"><span style="font-family:宋体;">二、资助条件：</span></span></strong></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">1、遵守国家法律、遵守《高等学校学生行为准则》和就读学校的各项规章制度、道德品质良好；</span></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">2、家庭经济困难、无力支付大学期间全部或部分学习和生活费用；</span></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">3、学习勤奋、成绩优良。申请人上学年必修课考试成绩排名在全年级前50%以内，若无排名，则学年必修课平均成绩在80分以上。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size:20px;"><strong><span style="font-family:宋体;">三、资助方案：</span></strong></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">1、获资助同学每人将获得3000元助学金；</span></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">2、获资助的同学需加入四川大学欧莱雅爱心社，由校团委组织爱心社同学常态化地开展公益服务，对受助同学成长成才提供各方面能力提升和锻炼。</span></span></div> <div> 	<span style="font-size:16px;"><span style="font-family:宋体;">3、校团委、中国青少年发展基金会和欧莱雅公司将从获得资助的同学中择优选拔，推荐参加四川大学欧莱雅海外交流项目。</span></span></div> <div> 	<span style="font-size:16px;"><span
