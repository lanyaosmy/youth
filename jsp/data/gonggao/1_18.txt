src="/index.php/system/web/view/viewnumber/id/14419"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">各级团组织、全校同学：</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　为贯彻实施高校基层团支部&ldquo;</span><span style="color:black;">活力提升</span><span style="color:black;">&rdquo;</span><span style="color:black;">工程，探索新形势下团建工作的新途径，发挥团员青年主体作用，不断提升增强基层团支部的活力，进一步增强基层团支部的凝聚力和战斗力、扩大团的活动的有效覆盖面，我校于2015</span>年10月正式启动四川大学基层团支部建设发展基金（以下简称团建基金），获得全校各级团组织和团员青年的广泛关注和积极参与，截止目前团建基金已成功结项6批共74个项目，发放基金95200元。团建基金从申报、审议、考核、综合评定到结项答辩，坚持规范管理，鼓励创新，以评促改，使我校基层团组织始终保持活力。根据《四川大学基层团支部建设发展基金管理办法》的相关要求，<span style="color:black;">现启动四川大学2017-2018</span>学年第一、二批基层团支部建设发展基金申报工作，具体工作通知如下：</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　<span style="font-family:黑体;">一、项目要求</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（一）团建基金申报、审批每一学年安排四次，每学期同时立项两批。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（二）团建基金管理按照&ldquo;突出重点、合理配置、专款专用&rdquo;的原则，根据活动形式和活动内容进行有区分性和倾向性的支持，积极鼓励基层团支部开展具有思想引领作用和有利于基层团组织建设的活动。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（三）团建基金项目形式不限，可涉及志愿公益、社会调研、学术探讨、文化体育、班团建设等，力求主题积极向上，能够充分调动团员积极性，促进团支部健康发展，反映当代大学生的思想动态，展现当代大学生的精神风貌，在一定程度上丰富校园文化、加强思想政治引领和价值引领。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　<span style="font-family:黑体;">二、项目评审</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（一）团支部申报</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">学院团委向本学院各年级团支书（若学院2017</span>级基层团支部还未选出团支书，则面向各班的临时负责人）详细介绍团建基金项目，鼓励各支部积极参与。各团支部即日起可按申报要求筹备项目方案，填写项目方案书、项目预算申请表向学院团委进行申报。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（二）学院初审</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">各学院团委对团支部申报的团建基金项目进行初审，筛选推报至少一个团建基金项目，出具《四川大学基层团支部建设发展基金项目学院评估意见表》（见附件）并注明申报等级。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　各学院团委于10</span>月18日24：00前提交推报的所有项目电子版材料（包括项目方案书、预算申请表及评估意见）发送至邮箱SCUtwzzbjc@163.com，校团委组织部将于10月20日统一收取加盖学院团委签章的《四川大学基层团支部建设发展基金项目学院评估意见表》。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（三）校团委综合评审</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　校团委将根据项目方案、预算申请表、评估意见等相关材料对项目主题及形式等进行初步审核。通过审核后，申报项目团支部进行现场展示，评议小组及参会人员（包括学院团委学生副书记、学院组织部部长、申报项目的团支书和一名普通团员）进行评审投票（现场展示具体要求及投票细则参见附件）。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　综合评审时间另行通知。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">　　（四）终审和资金发放</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span style="color:black;">校团委结合项目资料和综合评审情况，终审确定资助项目及等级，并发放资金。</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　<span style="font-family:黑体;">三、工作要求</span></span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><span
