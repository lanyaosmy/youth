<div class="content-art">  			<h1>关于组织开展“挑战杯”四川大学2017年大学生创业竞赛及预报名的通知</h1>  			<div class="attr">  				发布时间：2017-07-06 19:25:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14146"></script>  			</div>  			<div class="content-text">  				<div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">全校同学：</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &ldquo;创青春&rdquo;大学生创业大赛（简称&ldquo;小挑&rdquo;，原为&ldquo;挑战杯&rdquo;大学生创业计划竞赛）是由共青团中央、教育部、中国科协、全国学联主办的一项重要赛事，自1998年清华大学拉开帷幕以来，至今已近二十年的历史。大赛旨在鼓励和培养青年学生的创新创业精神，锻炼和提高青年学生的实践能力。多年来，我校学生项目屡次荣获国赛金奖等荣誉，在2016&ldquo;创青春&rdquo;全国大学生创业大赛中，荣获两项金奖、两项银奖、一项铜奖的优异成绩，捧回团体荣誉&ldquo;优胜杯&rdquo;。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;作为&ldquo;创青春&rdquo;大学生创业大赛的校内选拔赛，&ldquo;挑战杯&rdquo;四川大学学生科技节组委会决定于近期启动&ldquo;挑战杯&rdquo;四川大学2017年大学生创业竞赛，鼓励参赛同学充分利用暑期开展参赛组队、创业项目选择、技术研发、市场调研、创业方案撰写、营销推广、市场拓展等相关工作。为进一步提升学生参赛项目质量，增强学生团队创新创业能力和素质，并针对性地开展培训指导、项目孵化基金支持、申请入驻园区或创办公司等方面的帮扶工作，为2017年&ldquo;创青春&rdquo;大学生创业大赛选拔更优质的创新创业项目和团队，决定同时启动竞赛预报名工作。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;现将有关事宜通知如下：</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp;<strong>一、大赛内容</strong></span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;根据2016年&ldquo;创青春&rdquo;全国大学生创业大赛主体赛规程设置3项主体赛事：</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（一）创业计划类项目：</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;面向四川大学全日制在校学生（正式注册的全日制非成人教育学生），以商业计划书评审作为参赛项目的主要评价内容。参加竞赛项目分为已创业与未创业两类；分为A农林、畜牧、食品及相关产业，B生物医药，C化工技术和环境科学，D信息技术和电子商务，E材料，F机械能源，G文化创意和服务咨询等7个组别。实行分类、分组申报。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;未创业类： 拥有或授权拥有产品或服务，具有核心团队，具备实施创业的基本条件，但仅限于创业计划阶段，暂未开展创业实践。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;已创业类： 拥有或授权拥有产品或服务，已在工商、民政等政府部门注册登记；或者拥有或授权拥有产品或服务，已开展创业实践，有固定的创业场所，并稳定运营3个月以上，可申报已创业类（需提供单位概况、法定代表人情况、营业执照复印件、税务登记证复印件、组织机构代码复印件等材料作为作品文本附件）。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（二）创业实践类项目：</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;面向四川大学在校学生或毕业未满5年的四川大学毕业生和在校学生，以盈利状况、发展前景等作为参赛项目的主要评价内容。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;拥有或授权拥有产品或服务，并已在工商、民政等政府部门注册登记为企业、个体工商户、民办非企业单位等组织形式，且法人代表或经营者符合规定是四川大学在校学生或毕业未满5年的四川大学毕业生和在校学生，运营时间在3个月以上的项目，可申报该类别（需提供单位概况、法定代表人情况、营业执照复印件、税务登记证复印件、组织机构代码复印件等材料作为作品文本附件），申报不区分具体类别、组别。</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（三）公益创业类项目：</span></span></div> <div> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;面向四川大学在校学生（正式注册的全日制非成人教育学生），以创办非盈利性质社会组织的计划和实践等作为参赛项目的主要评价内容。拥有较强的公益特征（有效解决社会问题，项目收益主要用于进一步扩大项目的范围、规模或水平）、创业特征（通过商业运作的方式，运用前期的少量资源撬动外界更广大的资源来解决社会问题，并形成可自身维持的商业模式）、实践特征（团队须实践其公益创业计划，形成可衡量的项目成果，部分或完全实现其计划的目标成果）的项目。申报不区分具体类别、组别。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>二、校赛时间安排</strong></span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>校赛阶段：</strong></span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（一）2017年7月14号（周五），将进行四川大学2017年&ldquo;创青春&rdquo;全国大学生创业竞赛<strong>预报名</strong>，并根据团队的选题和前期工作评选<strong>孵化基金支持名额</strong>。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">预报名：</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">参赛团队提交作品申报书（见附件1）作为预报名。校团委将根据参赛团队预报名情况有针对性地进行培训指导、项目孵化基金支持。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="color:black;">：30-12：00，14</span>：30-18：00将纸质版申报书（2份）交至望江校区行政楼团委240&nbsp;<span style="color:black;">办公室</span>。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（二）2017年7至8月，各参赛队伍充分利用暑假时间，针对项目开展调研，完成初稿；</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（三）2017年9月，召开大赛系列宣讲会、培训会，进行竞赛指导工作；</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（四）2017年9至11月，各参赛队伍多寻求指导老师指导，开展产品、技术完善，市场营销推广；</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">（五）2017年12月，学校将面向全校征集作品申报书和创业方案，并组织校级选拔：包括作品文本初审和决赛答辩。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>三、参赛方式：</strong></span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;此次比赛以团队形式进行报名，原则上校赛阶段参赛队伍人数不超过8人，要求跨学科组成专业优势互补的团队，鼓励跨学历层次（创业计划竞赛不以团队成员最高学历层次来划分学生团队等级）。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;建议每支参赛团队具备项目技术、管理、市场营销、财务分析、法律等专业或特长的学生。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;若是技术类创业项目，建议找一名技术方面的指导老师，一名市场方面的指导老师。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;"><strong>四、作品要求：</strong></span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">1、要求参赛者组成优势互补的创业团队，以具有市场前景的产品或服务作为创业项目，围绕创业项目，完成一份完整、具体、深入，创新性、可行性、操作性俱佳的创业计划书/项目运营报告，并针对创业方案开展创业实践。创业计划书文本应基于创业项目，着眼于特定的市场、竞争、营销、运作、管理、财务等策略方案，描述公司的创业机会，描述把握这一机会创立公司的过程并说明所需资源。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">2、参赛项目可以是一项发明创造、专利技术，也可以是一项可能研发实现的概念产品或创新性的服务。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span style="font-family:楷体_gb2312;">3、作品申报数量要求：根据学院规模和专业情况，针对各学院设定最低任务指标，申报作品数量将纳入当年考核体系。</span></span></div> <div align="left"> 	<span style="font-size:18px;"><span
