<div class="content-art">  			<h1>关于开展2017年成都市“创业新星计划”的通知</h1>  			<div class="attr">  				发布时间：2017-09-12 00:04:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14251"></script>  			</div>  			<div class="content-text">  				<div align="center"> 	&nbsp;</div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">全校同学：</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;为全面贯彻落实成都市第十三次党代会精神，牢固树立&ldquo;不唯地域、不求所有、不拘一格&rdquo;的新人才观，根据《关于激励青年（大学生）创新创业 实施&ldquo;创业新星计划&rdquo;的意见（试行）》相关要求，激励和引导青年（大学生）在蓉创新创业，培育创新创业人才生力军，为建设全面体现新发展理念的国家中心城市汇聚青春力量，经成都市人才工作领导小组同意，将开展2017年成都市&ldquo;创业新星计划&rdquo;工作。我校团委现组织符合条件的同学参与该计划，有关事项通知如下：</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;"><strong>一、&ldquo;创业新星计划&rdquo;目标任务</strong></span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 坚持实践育才、创业兴市导向，在创办高成长性企业、具有开拓创新精神和竞争能力的青年（大学生）群体中，重点选拔和培育100名左右创业新星。</span></span></div> <div style="margin-left: 42pt;"> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;"><strong>二、遴选条件</strong></span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">1. 选拔对象&nbsp;</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;创新能力强、创业项目优、发展潜力大的我校在校生及毕业五年内的毕业生，应符合以下基本条件：</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（1）创办企业符合成都市产业发展方向，市场开发潜力大，具有一定技术、管理或商业模式的创新性</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（2）持有企业30%以上股权</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（3）创办企业主要纳税关系在成都市且连续纳税一年以上</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（4）创办企业为员工连续缴纳社保一年以上</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（5）创业团队成员获得大专及以上学历者占60%以上</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（6）个人及创业企业信用良好，无不良信用记录</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（7）创办企业年销售收入200万元以上</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">2.特别说明</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;对创业园、孵化器、加速器等平台内的企业，拥有自主知识产权的企业，3年内获得国家、省、市专项资助及获得风投资金的企业创办者优先考虑。特别优秀的青年（大学生）创业人才，可申请破格。以上所称&ldquo;特别优秀&rdquo;是指：</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（1）创业成效明显，在带动当地及周边群众共同创业、增加收入方面发挥较大作用</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（2）创业事迹突出，对同行业具有较大带动、引领和示范作用，具有一定社会影响力，在创业活动中起到带头示范作用</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（3）评审专家一致公认的优秀创业者。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;"><strong>三、遴选程序</strong></span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 2017年成都市&ldquo;创业新星计划&rdquo;评选工作在市人才工作领导小组领导、市人才工作领导小组办公室总体协调下开展，分为申报、初审、专家评审、审定、批准五个流程。</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（一）申报</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;申报截止时间为2017年9月25日。申报者须下载《2017年成都市&ldquo;创业新星计划&rdquo;申报书》（见附件），认真填写，准备相关佐证资料，<a href="mailto:%E5%B9%B6%E4%BA%8E2017%E5%B9%B49%E6%9C%8824%E6%97%A5%E5%89%8D%E5%8F%91%E9%80%81%E8%87%B3%E9%82%AE%E7%AE%B1thor.song@scucj.com">并于2017年9月25日前发送至邮箱thor.song@scucj.com</a>,邮箱命名格式为&ldquo;创业新星计划+申报人姓名+联系电话&rdquo;。</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（二）初审</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;初审时间为10月。由区（市）县委组织部牵头组织，区（市）县人社部门负责对申报者所在企业的社保缴纳情况进行审核，区（市）县经科部门负责对申报者所在企业的产业发展方向、市场潜力等情况进行审核，区（市）县工商部门负责对申报者所持企业股权以及企业注册情况进行审核，区（市）县税务部门负责对申报者所在企业的纳税情况进行审核，区（市）县金融机构负责对创业者及所在企业的征信情况进行审核。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（三）专家评审</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp; &nbsp; &nbsp; &nbsp;专家评委名单由市委组织部提供。团市委组织专家评委进行评审，并根据评审结果确定入选建议人选名单。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（四）审定</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 团市委会同市委组织部、市经信委、市科技局、市人社局根据评审意见提出拟入选人员名单，报市人才工作领导小组审定。</span></span></div> <div> 	&nbsp;</div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">（五）批准</span></span></div> <div> 	<span style="font-size: 18px;"><span style="font-family: 楷体_gb2312;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
