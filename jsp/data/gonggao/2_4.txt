<div class="content-art">  			<h1>关于认真做好四川大学江安校区西园围合寝室团支部换届选举的通知</h1>  			<div class="attr">  				发布时间：2017-09-27 18:35:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14396"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">各学院分团委：</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　为进一步加强我校基层团组织建设，健全团干部工作队伍，不断提高团组织吸引力、凝聚力、不断扩大团的工作有效覆盖面，为我校<span style="color:black;">深入贯彻落实</span>《高校共青团改革实施方案》，积极探索&ldquo;1+X&rdquo;团建模式，依托寝室围合建立团组织，构建&ldquo;多种模式、多重覆盖&rdquo;的团建创新机制奠定重要基础，按照《中国共产主义青年团基层组织选举规则》要求，现就认真做好江安校区西园围合寝室团支部换届选举工作相关事项通知如下：</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　一、组织设置原则</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　1</strong><strong>、团支部设置原则。</strong>根据团章和《中国共产主义青年团基层组织选举规则》有关规定，团员3人以上的单位应按建制单独设立团支部，江安校区西园寝室成员满足3人以上的，应建立寝室团支部。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　2</strong><strong>、团支部委员会设置原则。</strong>根据规定，支部委员会由团员大会选举产生，委员名额根据团员人数和工作需要确定，原则上不得超过或等于支部团员的半数，一般为3-5人。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　3</strong><strong>、支部委员设置原则。</strong>支委会可根据支部委员名额的多少和实际工作需要，设组织、宣传、实践等委员，委员可兼任。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　二、方法与步骤</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　本次换届选举工作于10月16日前完成，分5个步骤展开。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　1</strong><strong>、召开支部团员大会。</strong>组织团员认真学习团章，对团员进行民主集中制和团员权利、义务的教育，明确选举意义和基本要求，进一步统一思想、提高认识。按照组织设置原则，明确新一届支部委员会的名额，并充分酝酿委员候选人名单。团支部委员一般实行差额选举，委员候选人名额应多于应选名额的百分之二十。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　2</strong><strong>、召开选举大会。</strong>候选人报所在围合团委，确定人选后，各支部召开团员大会正式选举。会议一般由本届团支部书记主持，到会团员人数超过支部全体团员人数五分之四方可选举。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　3</strong><strong>、支部委员会召开第一次会议。</strong>选举大会结束后，新一届支部委员会推荐一名新当选的委员主持召开第一次会议，等额选举产生团支部书记、副书记，研究支部委员分工。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　4</strong><strong>、报告选举结果。</strong>各支部要尽快将选举情况，包括到会人数、当选的支部委员得票数、书记和委员名单及分工情况形成书面报告报所在围合团委批准。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;"><strong>　　5</strong><strong>、团委审批。</strong>围合团委批准新一届支部支委会后，将以书面形式正式批复并在青春川大官网进行公示，各支部要做好有关资料的整理归档工作。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　三、工作要求</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　1、寝室团支部换届选举工作是加强&ldquo;1+X&rdquo;团组织建设的一项重要内容，要在寝室所属学院团委、围合团委的指导以及班级团支部的协助下进行，各学院安排一名团委学生副书记专项负责本学院寝室团支部选举工作。按照团章规定的换届选举程序，坚持民主集中制原则，尊重和保障团员的民主权利，有步骤、有秩序地进行。</span></span></div> <div> 	<span style="font-family:仿宋_gb2312;"><span style="font-size:18px;">　　2、把好时间关。选举结束后，各寝室团支部要及时报送选举结果，确保换届选举工作按时顺利完成。团支部选举结果报告要在10月20日前报本围合团委，填写《寝室团支部委员、书记基本情况统计表》（见附件），纸质版在10月17日-20日晚19:00-21：00交至四舍活动室，电子版发围合分团委管理办公室邮箱：scuwgb@163.com。</span></span></div>
