<div class="content-art">  			<h1>商学院成功举行运动会动员大会</h1>  			<div class="attr">  				发布时间：2017-10-20 15:41:00  浏览次数：<script type="text/javascript" src="/index.php/system/web/view/viewnumber/id/14731"></script>  			</div>  			<div class="content-text">  				<div> 	<span style="font-family:宋体;"><span style="font-size:16px;">&nbsp; &nbsp; 2017年10月17日晚，由商学院团委学生会体育部主办的商学院运动会动员大会在江安校区学生西园六舍活动室顺利举行。四川大学商学院党委副书记兼纪委书记李小平老师、2016级辅导员唐建民老师、2017级辅导员吴悦老师，商学院团委学生会执行主席团、部长团部分同学以及本届校运会参赛运动员代表参加了此次动员大会。</span></span></div> <div> 	<span style="font-family:宋体;"><span style="font-size:16px;">&nbsp; &nbsp; 会议伊始，李小平书记简要介绍了自己分管的工作，并希望借此次座谈会的机会与大家相互认识、了解大家训练筹备中面临的实际困难，更好地引导、监督、服务同学们成长成才。</span></span></div> <div> 	<span style="font-family:宋体;"><span style="font-size:16px;">&nbsp; &nbsp; 李书记首先代表学院向大家的积极参与表示感谢。他指出，青年时期的毛泽东同志就十分重视体育运动，曾在《新青年》杂志上发表《体育之研究》一文，强调体育运动对于我们保持身体健康、促进思想成熟的重大意义；习近平同志更是一位运动迷，多次就健康中国建设发表重要讲话。环顾世界，海外名校也长期致力于体育运动的普及工作，形成了一大批极具学校特点、地域特色的活动。无数的例子向我们证明，体育运动对于个人、学校、国家发展极其重大的作用。在人民生活水平不断提高的当下，提高生活品质的一个重要环节就在于积极参与体育健身活动。</span></span></div> <div> 	<span style="font-family:宋体;"><span style="font-size:16px;">李书记强调，学校、学院始终高度重视体育事业建设。徐玖平院长曾就全民健身工作、公共体育设施开放等事务向俞正声同志建言献策；校运会更是由校领导亲自主持，是学校最重大的校园活动之一，更是展现各学院组织动员能力、学生精神风貌的一次大阅兵。</span></span></div> <div> 	<span style="font-family:宋体;"><span style="font-size:16px;">&nbsp; &nbsp; 李书记要求，各位同学一定要从思想上高度重视本次校运会工作，积极参与前期筹备工作，大力弘扬&ldquo;团结、友谊、和平&rdquo;的精神，重在参与，展现风采；要尽快调整状态，以健康的生活方式、科学的训练模式为校运会的到来做好充足的准备，在大赛中正常发挥应有水平。</span></span></div> <div> 	<span style="font-size: 16px; font-family: 宋体;">&nbsp; &nbsp; 李书记的讲话引起了在座的同学的强烈共鸣，大家十分感谢学院一直以来对学生培养工作的高度重视。部分学生代表还就一些具体问题发表了意见。会后，李书记还前往十九舍慰问了正在积极备战运动会的运动员们。</span></div> <div style="text-align: center;"> 	<span style="font-size: 16px; font-family: 宋体;"><img alt="" src="/youth/uploads/up/default/-B4-F3-BC-D2-C8-CF-D5-E6-F1-F6-CC-FD-C0-EE-D0-A1-C6-BD-CA-E9-BC-C7-BD-B2-BB-B0.jpg" style="width: 700px;
